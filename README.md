# Quantitative Trading Strategies Backtester

A lightweight Python framework designed for backtesting mean-reversion and statistical arbitrage strategies on high-frequency data.

## Features
- **Data Ingestion:** Optimized pipeline for processing large CSV datasets.
- **Signal Generation:** Implementation of Bollinger Bands and RSI strategies logic.
- **Risk Management:** Kelly Criterion implementation for position sizing.

## Tech Stack
- Python 3.9+
- NumPy (Vectorized calculations)
- Pandas (Time-series manipulation)
- Matplotlib (Visualization)
